HTML

It is the standard markup language for creating and structuring web content, providing a set of elements and tags that define the structure and layout of a webpage.

CSS

CSS, which stands for Cascading Style Sheets, is a styling language used to control the presentation and layout of HTML documents. It enables web developers to define the visual aspects of a website, such as fonts, colors, spacing, and responsive design, ensuring a consistent and appealing user experience across different devices.

Java Script

JavaScript (JS) is a versatile and widely-used programming language primarily employed for creating dynamic and interactive web content. It runs on the client side, enabling enhanced user experiences by manipulating HTML, CSS, and handling asynchronous operations.

Bootstrap

Bootstrap is a popular open-source front-end framework that facilitates the development of responsive and mobile-first websites. Developed by Twitter, Bootstrap provides a collection of pre-designed components and utilities, streamlining the process of creating visually appealing and consistent user interfaces.

Java

Java is a versatile, object-oriented programming language known for its platform independence, allowing developers to write code that can run on various devices. It's widely used in building robust and scalable applications, from mobile and web development to enterprise-level systems.

SQL

Structured Query Language (SQL) is a powerful domain-specific language used for managing and manipulating relational databases. SQL enables users to retrieve, insert, update, and delete data, making it a fundamental tool for database management and interaction.
